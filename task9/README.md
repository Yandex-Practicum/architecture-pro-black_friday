# Задание 9. Настройка чтения с реплик и консистентность

## Операции чтения

### products

| Операция | Откуда читать | Допустимая задержка | Причина |
|----------|---------------|---------------------|---------|
| Поиск товаров по категории | Secondary | 2-5 сек | Не критично показать старую цену |
| Фильтрация по цене | Secondary | 2-5 сек | Допустимо отставание |
| Описание товара | Secondary | 2-5 сек | Редко обновляется |
| Проверка остатка перед покупкой | Primary | 0 сек | Риск продажи несуществующего товара |
| Обновление остатка | Primary | 0 сек | Критично для корректности |


### orders

| Операция | Откуда читать | Допустимая задержка | Причина |
|----------|---------------|---------------------|---------|
| История заказов | Secondary | 5-10 сек | Не критично увидеть последний заказ с задержкой |
| Статус заказа | Primary | 0 сек | Пользователь ждет актуальный статус |
| Создание заказа | Primary | 0 сек | Критично для корректности |
| Список заказов админа | Secondary | 5-10 сек | Аналитика, задержка допустима |


### carts

| Операция | Откуда читать | Допустимая задержка | Причина |
|----------|---------------|---------------------|---------|
| Получение корзины | Primary | 0 сек | Пользователь видит свои изменения сразу |
| Добавление товара | Primary | 0 сек | Изменение должно быть видно сразу |
| Удаление товара | Primary | 0 сек | Изменение должно быть видно сразу |
| Слияние корзин (гость → user) | Primary | 0 сек | Критично не потерять товары |



## Настройка в приложении

```javascript
// products: поиск и каталог
db.products
  .find({ category: "electronics" })
  .readPref('secondaryPreferred')

// products: проверка остатка
db.products
  .findOne({ _id: productId }, { stock: 1 })
  .readPref('primary')

// orders: история
db.orders
  .find({ user_id: userId })
  .readPref('secondaryPreferred')

// orders: статус
db.orders
  .findOne({ _id: orderId }, { status: 1 })
  .readPref('primary')

// carts: всегда primary
db.carts
  .find({ user_id: userId, status: "active" })
  .readPref('primary')
```
