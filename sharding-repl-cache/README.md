# Mongo Sharding (With Replica Sets)

Для запуска и проверки работы шардирования в MongoDB с репликацией выполните следующие шаги:

1. Поднимите контейнеры в фоновом режиме:
   ```bash
   docker compose up -d
   ```

2. Запустите скрипт инициализации. Он настроит Config Server (1 узел), оба шарда (по 3 узла в каждом), добавит их в кластер и наполнит тестовыми данными (коллекция `helloDoc`):
   ```bash
   sh init.sh
   ```

3. Запустите проверочный скрипт, чтобы убедиться в корректном распределении данных по обоим шардам:
   ```bash
   sh check.sh
   ```
