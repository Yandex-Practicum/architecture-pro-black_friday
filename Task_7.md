## Решение по шардированию

### orders

#### Основные опелации по коллекции: 
- Быстрое создание заказов с одновременным списанием остатков.
- Поиск истории заказов конкретного пользователя.
- Отображение статуса заказа.

#### Решение

Нужно применить хеш-шардирование по свойству *user_id* . 

- Основная операция: поиск истории заказов конкретного пользователя. 
- Данное шардирование позволит локализовать все заказы одного пользователя в рамках одного шарда.  


### products
#### Основные операции по коллекции
- Частые обновления остатков при покупках.
- Поиск товаров по категориям и фильтрация по диапазону цен.
- Описание товара на странице продукта.

#### Решение
Нужно применить хеш-шардирование по свойству *category*

- Основная операция: Поиск товаров по категориям и фильтрация по диапазону цен.
- Данный подход к шардированию позволит локализовать товары в рамках одного шарда по категории товара
- Поиск по диапозону цен обычно выполняется в рамках какой-то конкретной категории

### carts
#### Основные операции по коллекции
- Создание корзины, когда заходит гость или новый пользователь.
- Получение текущей корзины по фильтру { session_id, status:"active" } или { user_id, status:"active" }.
- Добавление или замена товара в корзине.
- Удаление товара из корзины.
- Слияние гостевой корзины в пользовательскую, если пользователь залогинится:
   - прочитать гостевую { session_id, status:"active" };
   - добавить её items в корзину { user_id, status:"active" };
   - отметить гостевую как abandoned.
- Отметка корзины как заказанной.

#### Решение
Нужно применить хеш-шардирование по составному ключу *user_id - session_id*

- Основная операция:  Получение текущей корзины по фильтру { session_id, status:"active" } или { user_id, status:"active" }.
- Данный подход к шардированию позволит равномерно распределить корзины по шардам, в зависимости от пользователей и их сессий
- Такой подход позволит так же выполнить условие с группировкой корзин в пользователекую в случае, если был выполнен вход в систему после заполнения корзины, без пересоздания или дополнительных действий с корзиной
