# Mongo Sharding

Для запуска и проверки работы шардирования в MongoDB выполните следующие шаги:

1. Поднимите контейнеры в фоновом режиме:
   ```bash
   docker compose up -d
   ```

2. Запустите скрипт инициализации. Он настроит Config Server, оба шарда, добавит их в кластер и наполнит тестовыми данными:
   ```bash
   sh init.sh
   ```

3. Запустите проверочный скрипт, чтобы убедиться в корректном распределении данных по обоим шардам:
   ```bash
   sh check.sh
   ```
