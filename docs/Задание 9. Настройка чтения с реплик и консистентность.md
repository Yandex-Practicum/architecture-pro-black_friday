## Таблица операций чтения: primary vs secondary


| Коллекция | Операция                                                                     | Куда читать | Допустимая задержка реплики |
| ------------------ | ------------------------------------------------------------------------------------ | --------------------- | ---------------------------------------------------- |
| **products**       | Страница товара (описание, фото)                           | Secondary             | до 30 сек                                       |
| **products**       | Остаток товара перед добавлением в корзину      | **Primary**           | —                                                   |
| **products**       | Поиск по категории / фильтр по цене                      | Secondary             | до 30 сек                                       |
| **orders**         | История заказов пользователя                               | Secondary             | до 10 сек                                       |
| **orders**         | Текущий статус заказа (страница отслеживания) | **Primary**           | —                                                   |
| **orders**         | Создание заказа (проверка перед записью)           | **Primary**           | —                                                   |
| **carts**          | Получение текущей активной корзины                    | **Primary**           | —                                                   |
| **carts**          | Слияние гостевой корзины с пользовательской   | **Primary**           | —                                                   |
| **carts**          | Аналитика брошенных корзин                                   | Secondary             | до 60 сек                                       |

## Допустимая задержка репликации


| Коллекция | Операция                               | Допустимая задержка |
| ------------------ | ---------------------------------------------- | ------------------------------------- |
| products           | Описание / поиск                  | до 30 сек                        |
| products           | Остаток (перед покупкой)   | 0 — только Primary             |
| orders             | История заказов                  | до 10 сек                        |
| orders             | Актуальный статус заказа | 0 — только Primary             |
| carts              | Активная корзина                | 0 — только Primary             |
| carts              | Аналитика abandoned                   | до 60 сек                        |

---

## Обоснование выбора

**products — остаток товара → Primary** Частые обновления при каждой покупке. Чтение с Secondary при задержке реплики = риск продажи товара, которого уже нет. Бизнес-риск критичный: overselling, возвраты, штрафы.

**products — описание, поиск → Secondary** Данные меняются редко (контент-менеджер). Задержка 30 сек. пользователю незаметна. Разгружает Primary на операции записи.

**orders — статус → Primary** Пользователь только что оплатил и смотрит статус. Устаревший статус — негативный UX и обращения в поддержку. Консистентность обязательна.

**orders — история → Secondary** История не меняется после завершения заказа. Задержка 10 сек. для списка прошлых заказов некритична.

**carts — активная корзина → Primary** Корзина обновляется постоянно: добавление товара, изменение количества, слияние при логине. Чтение устаревшей версии → потеря товаров из корзины или дублирование. Консистентность обязательна.

**carts — аналитика abandoned → Secondary** Читается фоновыми процессами, не пользователем. Точность до минуты достаточна для аналитики.
