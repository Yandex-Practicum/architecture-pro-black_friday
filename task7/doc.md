2. Потенциальные кандидаты для шард-ключей:
  - products:
	  - { category: 1 }
    - { category: 1, _id: "hashed" }
    - { "stock_by_geo.geo_zone": 1 } 
    - { category: 1, "stock_by_geo.geo_zone": 1 } 
    - { _id: "hashed" }
  - orders:
    - { user_id: 1 }
    - { geo_zone: 1, created_at: 1 }
    - { user_id: 1, created_at: 1 } 
    - { _id: "hashed" }
  - carts:
    - { _id: "hashed" }
    - { user_id: "hashed" }
    - { session_id: "hashed" }

3. Выбранная стратегия шардирования коллекций:
  - products: { category: 1, _id: "hashed" }
    Категория задаёт логическую группировку, а hash _id  внутри категории убирает «плотность» по одному шарду.
    Частые обновления остатков при покупках - идут по product_id (то есть _id), поэтому shard key, содержащий хэш _id, даст таргетирование на один шард и равномерное распределение обновлений.
    Поиск по category  сможет таргетить небольшой набор чанков, а не весь кластер. 
    Поиск товаров по категориям и диапазону цен - фильтр включает category и price, поэтому полезно иметь category частью ключа для локализации данных одного каталога.
  - orders: { user_id: 1 }
    Пользователей обычно много, что даёт равномерное распределение заказов по шардам.
    Все заказы пользователя на одном шарде, удобно для аналитики по пользователю, истории, повторных заказов.
    Поиск истории заказов конкретного пользователя - почти все запросы включают user_id, значит запросы будут таргетиться в один шард.
  - carts: { _id: "hashed" }
    Максимально равномерное распределение всех операций записи и обновления (добавление/удаление товаров) по шардам.
    Создание корзины - очень частые записи, причём в момент пиковых нагрузок (распродажи), важно равномерно распределить записи по шардам.
    Каждая корзина само-достаточна; редко выполняются операции, которые требуют «сканировать» много корзин (долговременная аналитика по корзинам обычно выносится в отдельную коллекцию).
4. Примеры команд MongoDB:
```js
sh.enableSharding("mobile_world");

sh.shardCollection(
  "mobile_world.products",
  { category: 1, _id: "hashed" }
);

sh.shardCollection(
  "mobile_world.orders",
  { user_id: 1 }
);

sh.shardCollection(
  "mobile_world.carts",
  { _id: "hashed" }
);
```
