# Выбор индексов

## Коллекция `carts`

Поле `owner_key` — это унифицированный идентификатор владельца корзины.
Поле используется для объединения логики гостевых и пользовательских корзин 
в одной коллекции и одном shard-ключе.

Формат:
```js
"u:<user_id>"     // корзина авторизованного пользователя
"s:<session_id>"  // гостевая корзина
```
* `{ owner_key: 1, status: 1 }` — быстрый поиск активной корзины пользователя или гостя.
* `{ expires_at: 1 }` (TTL) — автоматическая очистка устаревших корзин.
* `{ updated_at: 1 }` — оптимизация сортировки и частых обновлений корзины.

### Создание индексов

```js
db.createCollection("carts");
db.carts.createIndex(
    { owner_key: 1, status: 1 },
    { name: "idx_owner_key_status" }
);
db.carts.createIndex(
  { expires_at: 1 },
  { name: "idx_expires_at_ttl", expireAfterSeconds: 0 } // TTL: удалять ровно после expires_at
);
db.carts.createIndex(
    { updated_at: 1 },
    { name: "idx_updated_at" }
);
```

## Коллекция `orders`

* `{ order_id: 1 }` (unique) — уникальный бизнес-идентификатор заказа.
* `{ customer_id: 1, created_at: -1 }` — получение истории заказов с сортировкой по дате.
* `{ customer_id: 1, status: 1 }` — фильтрация заказов клиента по статусу.

### Создание индексов

```js
db.createCollection("orders");
db.orders.createIndex(
    { order_id: 1 },
    { name: "idx_order_id_unique", unique: true }
);
db.orders.createIndex(
    { customer_id: 1, created_at: -1 },
    { name: "idx_customer_created_at_desc" }
);
db.orders.createIndex(
    { customer_id: 1, status: 1 },
    { name: "idx_customer_status" }
);
```

## Коллекция `products`

* `{ category: 1, price: 1 }` — поиск товаров по категории и диапазону цен.
* `{ stocks.geo_zone: 1 }` — операции, связанные с региональной доступностью товаров.

### Создание индексов

```js
db.createCollection("products");
db.products.createIndex(
    { category: 1, price: 1 },
    { name: "idx_category_price" }
);
db.products.createIndex(
    { "stocks.geo_zone": 1 },
    { name: "idx_stocks_geo_zone" }
);
```

# Стратегии шардирования

Схема см. `./collections.plantuml`

---

## Коллекция `orders`
Shard key:
```js
{ customer_id: "hashed" }
```

Коллекция `orders` содержит заказы клиентов и имеет долгоживущий характер хранения данных.
Ключевые операции:
* получение истории заказов конкретного пользователя;
* отображение статуса заказа;
* создание новых заказов с высокой частотой.

Использование `customer_id` в качестве шард-ключа позволяет:
* эффективно обрабатывать запросы истории заказов, так как они адресуются к конкретному клиенту;
* избежать концентрации нагрузки на одном шарде, поскольку хеш-шардирование равномерно распределяет клиентов;
* заказы одного клиента на одном шарде;
* масштабировать систему горизонтально при росте числа пользователей.

---

## Коллекция `products`
Shard key:
```js
{ category: 1 }
```
Коллекция `products` активно используется для:
* поиска товаров по категориям;
* фильтрации по диапазону цен;
* частых точечных обновлений остатков по конкретному товару.

---

## Коллекция `carts`
Shard key:
```js
{ owner_key: "hashed" }
```

Коллекция `carts` хранит временные данные корзин как для гостей, 
так и для авторизованных пользователей.

Основные операции:
* получение активной корзины по `session_id` или `user_id`;
* частые обновления состава корзины;
* автоматическая очистка устаревших корзин (TTL).

Поле `owner_key` представляет собой унифицированный идентификатор владельца корзины 
(`user_id` или `session_id`) и позволяет:

* использовать единый шард-ключ для всех типов корзин;
* эффективно обрабатывать запросы получения активной корзины;
* равномерно распределять данные за счёт хеш-шардирования;
* избежать разделения коллекции по разным шард-ключам для гостей и пользователей.

TTL-индекс по полю `expires_at` работает независимо от стратегии шардирования и не влияет 
на выбор шард-ключа.


### Создание шардов
```js
sh.enableSharding("shop")
sh.shardCollection("shop.carts", { owner_key: "hashed" })
sh.shardCollection("shop.orders", { customer_id: "hashed" })
sh.shardCollection("shop.products", { category: 1 })
```
